# VoidUI

WindUI / Cascade style Roblox UI library (dark + lime).

## Load

```lua
local VoidUI = loadstring(game:HttpGet("https://raw.githubusercontent.com//VoidUI/main/VoidUI.lua"))()
```

## Demo

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com//VoidUI/main/VoidUI.Example.lua"))()
```

RightShift = show/hide.
